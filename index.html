🌐 CSPM (Cloud Security Posture Management)

교보DTS 기업 협업으로 진행한 AWS 자원 취약점 점검 프로젝트
  • GitHub : https://github.com/dev-donggyu/cspm
  • 프로젝트 발표 자료 :
  • 작업 참여 기간 : 2024.04 ~ 2024.06
  • 기술 스택 : Java, Spring Boot, Spring Data JPA, AWS-SDK, JSON, GitLab, Notion, Jira 등
  • 담당 파트 : Back-End & 인프라 구축
  • 담당 업무 :
    • 사용자 계정 정보와 AWS 자원 조회 정보 제공 API 구현
    • 비동기 방식을 이용한 AWS 자원 조회
      • 동기 방식과 비동기 방식의 성능 비교

        ⬆️ 동기 조회 방식

        🔼 비동기 조회 방식

    • AES256 알고리즘을 사용한 암호화 작업
      • AWS Key의 암호화 / 복호화를 사용한 결과
    • 다중 DataSource 방식을 사용한 데이터베이스 분리 작업
      • 서버 실행 시 생성되는 데이터베이스 목록
        • cspm : 계정 및 자원 조회 데이터 저장
        • comliance : 자원 취약점 히스토리 저장
    • Dev / Release 설정 파일 분리 작업
